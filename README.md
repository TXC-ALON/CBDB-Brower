# CBDB Explorer

基于 `React + Vite + Electron + better-sqlite3` 的离线桌面应用，用于浏览 CBDB 中国历代人物传记数据库。

## 启动开发

```bash
npm install
npm run dev
```

默认会尝试加载项目根目录下的 `cbdb_20260221.sqlite3`，也可以在应用左侧点击“选择数据库文件”手动指定。

## 打包 Windows 安装包

```bash
npm run build
```

构建产物在 `release/` 目录。

## 已实现功能

- 人物搜索：姓名（中文/拼音）+ 朝代 + 官职 + 科举条件，分页与关键字高亮
- 人物详情：基本信息、官职履历、家族关系、科举记录、社会关系
- 关系图谱：家族关系/社会关系网络可视化
- 地理分布：全国地图点位与人物迁徙轨迹（支持时间范围和朝代筛选）
- 统计分析：朝代分布、科举趋势、官职分布、家族网络规模
